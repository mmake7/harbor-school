## 🎯 왜 만들었을까

> *"혼자 살면서 가장 힘든 건 결정할 사람이 나밖에 없다는 것"*

도시에 혼자 사는 30대 IT/금융 전문직.
퇴근하고 냉장고 앞에서 고민하고,
통장 잔고 보며 한숨 쉬고,
밤늦게 SNS 들어가서 남들 글 읽다 잠든다.

**혼자사는 삶**은 이런 1인 가구의 일상을 AI가 조용히 챙기는 플랫폼입니다.

harbor.school 4주차 수업 퀘스트 6개를 각각 만들 수도 있었지만,
**"1인 가구의 하루를 관통하는 하나의 플랫폼"** 으로 기획·통합했습니다.

## ✨ 핵심 기능 한눈에

### 🏠 홈 — AI 시간대별 인사말

> 아침엔 활기차게, 저녁엔 따뜻하게.

Claude AI가 매일 4번(아침/오후/저녁/새벽) 다른 메시지를 생성합니다.
- 시간대에 맞는 헤드라인 + 서브라인 + CTA 버튼
- DB 캐싱으로 빠른 로딩 (하루 1회 생성, 이후는 즉시)
- 오후엔 "숨 좀 고를게요", 저녁엔 "오늘도 수고했어요"

### 🍳 식생활 — 냉장고 재료로 AI 레시피

> 퇴근하고 냉장고 열어서 "오늘 뭐 해먹지..."
> → Claude AI가 3가지 레시피를 제안합니다.

- **🧊 냉장고**: 재료 관리 (CRUD) + 유통기한 D-day 알림
- **🍳 만들기**: Claude API로 3종 레시피 자동 생성
  - 초간단 ver. (5-10분)
  - 든든한 ver. (15분)
  - 색다른 ver. (창의적)
- **📚 기록**: 생성된 레시피 영구 저장
- **🎁 보너스**: Notion DB 자동 업로드 (외부 공유용)

### 💬 소셜 — 혼삶톡 익명 게시판

> "이런 얘기 어디 하기 그래서..."
> → 익명으로 털어놓고, 따뜻한 반응 받기.

- **4개 카테고리**: 고민 / 칭찬 / 응원 / 수다
- **3종 반응**: 💙 공감 / 🤗 응원 / ✨ 칭찬 (토글 지원)
- **자동 익명 닉네임**: "다정한 부엉이 #4321"
- **답글 + 베스트 게시글** 기능

### 🎮 밸런스 — 혼삶 vs 딜레마 투표

> "월세 vs 전세, 둘 다 답 없다"
> → 투표하고 퍼센티지 실시간 확인.

- **5개 카테고리**: 💰돈 / 🍜음식 / 🏠주거 / 💼일 / 🎮여가
- **실시간 집계**: 투표 즉시 A/B 비율 애니메이션
- **인기 질문 랭킹 TOP 5** + 나만의 질문 등록

### 📊 라이프 — 연봉/지출 익명 벤치마크

> "남들은 월급 얼마 받지, 얼마 쓰지..."
> → 30명 실제 데이터 대비 내 등급 확인.

- **8개 지출 카테고리**:
  🏠주거 · 🍽️식비 · 🚗교통 · 📱통신비 · 📺구독료 · 👔쇼핑 · 🐾반려동물 · 🎮여가
- **게임식 등급 시스템**:
  💎 다이아 (상위 10%) / 🥇 골드 (30%) / 🥈 실버 (50%) / 🥉 브론즈 (70%)
- **직군별/연령별 필터** + 저축률 자동 계산

### 🔮 운세 — 혼삶 AI 점성술사 (보너스)

> 아침에 한 번 열어보는 재미.

Claude AI가 매일 개인화된 운세를 생성합니다.
- **4개 혼삶 테마**:
  🍚 혼밥운 / 🏠 자취운 / 💰 저축운 / 💝 만남운
- **오늘의 행운 아이템** + **오늘의 한마디**
- **하루 1회 제한** (세션별 · UNIQUE 제약)
- **지난 7일 기록** 타임라인

## 🤖 진짜 AI 앱

이 앱은 **목업이 아닙니다.**
실제 Anthropic Claude API를 호출하는 완전 동작 AI 앱입니다.

### 🎯 Claude API 활용 3곳

#### 1. 레시피 생성 (식생활)
냉장고 재료 → Claude Sonnet 4 → **3가지 맞춤 레시피 JSON**
- 칼로리·시간·난이도 자동 계산
- 대체 재료 추천
- Notion DB 자동 업로드

#### 2. 오늘의 운세 (운세)
오늘 날짜 + 세션 ID → Claude Sonnet 4 → **4테마 개인 운세**
- 점수(1-5) + 테마별 메시지
- 행운 아이템 + 조언
- 하루 1회 제한 (DB UNIQUE 제약)

#### 3. 랜딩 카피 (홈)
현재 시간대 → Claude Sonnet 4 → **시간대별 감성 카피**
- 아침/오후/저녁/새벽 4가지 무드
- DB 캐싱으로 하루 1회만 생성 (비용 절감)

### 📊 비용 최적화

- **캐싱 전략**: 동일 키 요청은 DB에서 즉시 반환
- **모델 선택**: Sonnet 4 (속도 + 품질 균형)
- **토큰 제한**: max_tokens 1024~2048
- **Fallback 로직**: AI 실패 시 정적 대안 제공

## 🧑‍💻 개발 과정

### 🤖 Built with Claude Code

전 과정을 AI 에이전트와 협업으로 진행했습니다.

**개발 플로우:**
기획 (Claude 대화)
↓
설계 (DB 스키마 + API 구조)
↓
구현 (Claude Code 실시간 코딩)
↓
디버깅 (실시간 에러 로그 분석)
↓
배포 (Vercel 자동 CI/CD)
↓
개선 (사용자 피드백 반영 반복)

### 주요 의사결정 포인트

- **Skill vs Web**: Q1은 Skill로, Q2~Q7은 Web으로 (용도 분리)
- **통합 vs 분리**: 6개 퀘스트를 **하나의 플랫폼**으로 통합 선택
- **Vercel 12개 한도**: 새 API 대신 query 분기 (`?view=...`) 활용
- **AI 비용 최적화**: DB 캐싱 + 하루 1회 제한
- **디자인 일관성**: Linear 모노톤 + 인주색 포인트
