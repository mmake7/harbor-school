HTML 기본 태그
5가지.

웹 페이지의 뼈대를 만드는
가장 핵심적인 태그를 알아봅니다.

<h1> <p> <a> <img> <ul>
스크롤하여 시작

Tag 01

<h1> ~ <h6>

페이지의 제목과 구조를 정의하는 핵심 태그

index.html 
<h1>가장 큰 제목</h1>
<h2>두 번째 제목</h2>
<h3>세 번째 제목</h3>
Result

가장 큰 제목

두 번째 제목

세 번째 제목

✓ h1은 페이지당 하나만 사용하는 것이 SEO에 유리합니다.
✓ 검색엔진이 제목 태그를 통해 페이지 구조를 파악합니다.

Tag 02

<p>

본문 텍스트를 담는 가장 기본적인 태그

index.html 
<p>첫 번째 문단입니다.</p>

<p>두 번째 문단입니다.
   줄바꿈은 무시됩니다.</p>
Result

첫 번째 문단입니다.

두 번째 문단입니다. 줄바꿈은 무시됩니다.

✓ HTML에서 줄바꿈은 <br> 태그를 사용해야 합니다.
✓ <p> 태그는 블록 요소로, 자동으로 위아래 여백이 생깁니다.

Tag 03

<a>

다른 페이지나 사이트로 이동하는 하이퍼링크

index.html 
<a href="https://naver.com">
  네이버로 이동
</a>

<a href="page2.html"
   target="_blank">
  새 탭에서 열기
</a>
Attributes
href 이동할 URL 주소
target="_blank" 새 탭에서 열기
target="_self" 현재 탭에서 열기 (기본값)
✓ 외부 링크에는 보안을 위해 rel="noopener"를 함께 사용합니다.

Tag 04

<img>

웹 페이지에 이미지를 삽입하는 태그

index.html 
<img
  src="photo.jpg"
  alt="사진 설명"
  width="300"
  height="200"
>
Attributes
src 이미지 파일 경로 (필수)
alt 대체 텍스트 — 접근성 (필수)
width / height 이미지 크기 지정
! alt 속성은 시각장애인 스크린리더와 SEO에 매우 중요합니다.
✓ <img>는 닫는 태그 없이 단독으로 사용하는 빈 요소(void element)입니다.

Tag 05

<ul> / <li>

항목을 나열할 때 사용하는 목록 태그

순서 없는 목록 
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
Result

<ul> 순서 없는 목록

  • HTML
  • CSS
  • JavaScript

<ol> 순서 있는 목록

  1. 첫 번째
  2. 두 번째
  3. 세 번째
✓ <ul>은 순서 없는 목록(Unordered), <ol>은 순서 있는 목록(Ordered)입니다.
✓ 네비게이션 메뉴도 <ul> + <li>로 만드는 경우가 많습니다.
