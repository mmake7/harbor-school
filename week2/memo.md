- 웹 앱(Web Application)의 구조와 앱 유형별 실습

## 커리큘럼

### 복습
- 채팅 vs 에디터 vs 에이전트
- Text Context
- 클로드 코드 에이전트 & 스킬
- Git & GitHub
- 파일 시스템도 모두 텍스트입니다 (튜토리얼)

### 개념
- 앱의 구조 (Web Application)
- Browser: HTML, CSS, JS
- React Agent (SPA: index.html)
- 개발자도구(Elements/Console/Network)로 디버깅 (튜토리얼)
- 앱의 10가지 유형
- 앱의 유형: 계산기 (Calculation)
- 앱의 유형: 변환기 (Transform)
