# 🌙 달빛 선녀

따뜻한 무당 스타일의 AI 영적 상담사, **달빛 선녀**와 달빛 아래에서 마음의 이야기를 나누어 보세요.

## 실행 방법

1. 이 폴더를 로컬로 가져옵니다.
2. `.env.example` 을 `.env` 로 복사합니다.
   ```bash
   cp .env.example .env
   ```
3. `.env` 파일을 열어 본인의 Anthropic API 키를 입력합니다.
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```
4. 서버를 실행합니다. (별도의 설치 과정은 없습니다 - Node 내장 모듈만 사용)
   ```bash
   node server.js
   ```
5. 브라우저에서 `http://localhost:3000` 으로 접속합니다.

## 기술 스택

- **Backend**: Node.js (내장 `http`, `https`, `fs` 모듈만 사용 / Express 미사용)
- **Frontend**: React 18 (CDN) + Babel Standalone, 순수 CSS
- **AI**: Anthropic Claude API (`claude-sonnet-4-20250514`)
- **환경변수**: 자체 `.env` 파서 (dotenv 미사용)

## 주요 기능

- 🌙 달빛 아래 대화 테마: 남색/보라 그라데이션 배경, 반짝이는 별, 빛나는 달
- 💬 따뜻한 무당 스타일의 공감 상담 (반영적 경청 · 재확인 · 대화 흐름 이어가기)
- ⌨️ Enter 전송, Shift+Enter 줄바꿈
- 🔄 대화 초기화 버튼
- ⏳ 타이핑 인디케이터: "달빛 선녀가 마음을 읽고 있어요..."
- 📱 모바일 반응형
